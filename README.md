## معرفی پروژه

یک سامانه خرید و فروش کالای دست دوم (مشابه دیوار) شامل دو بخش:

- **backend**: سرور REST با Spring Boot (پورت 8080)
- **frontend**: برنامه دسکتاپ با JavaFX

ارتباط دو بخش از طریق HTTP و JSON انجام می‌شود و احراز هویت با توکن JWT است.

## پیش‌نیازها

- Java 17 یا بالاتر
- Maven 3.8 یا بالاتر

## نحوه اجرا

### ۱) اجرای بک‌اند

```bash
cd backend
mvn spring-boot:run
```

سرور روی آدرس `http://localhost:8080` بالا می‌آید. در اولین اجرا، فایل دیتابیس `secondhand.db` ساخته می‌شود و داده‌های اولیه (کاربر مدیر، شهرها و دسته‌بندی‌ها) به صورت خودکار ثبت می‌شوند.

### ۲) اجرای فرانت‌اند



```bash
cd frontend
mvn javafx:run
```

## ذخیره‌سازی داده‌ها

- به صورت پیش‌فرض از **SQLite** استفاده می‌شود (فایل `secondhand.db` کنار پروژه backend) و نیازی به نصب چیزی نیست.
- در صورت تمایل به PostgreSQL، ابتدا دیتابیس و کاربر زیر را بسازید و سپس با پروفایل postgres اجرا کنید:

```sql
CREATE DATABASE secondhand_db;
CREATE USER secondhand_user WITH PASSWORD 'secondhand_password';
GRANT ALL PRIVILEGES ON DATABASE secondhand_db TO secondhand_user;
```

```bash
mvn spring-boot:run -Dspring-boot.run.profiles=postgres
```

- تصاویر آگهی‌ها در پوشه `uploads/` کنار بک‌اند ذخیره می‌شوند.

## امکانات پیاده‌سازی شده

### امکانات اصلی

- ثبت‌نام و ورود کاربران با اعتبارسنجی کامل (نام کاربری، رمز، موبایل، ایمیل)
- احراز هویت با توکن JWT و ذخیره رمز عبور به صورت هش (SHA-256)
- ثبت آگهی با عنوان، توضیحات، قیمت، وضعیت کالا، دسته‌بندی، شهر و تصویر
- گردش کار تایید آگهی توسط مدیر (در انتظار تایید / فعال / رد شده با ذکر دلیل / فروخته‌شده / حذف‌شده)
- ویرایش و حذف آگهی توسط صاحب آن (بعد از ویرایش، آگهی دوباره نیاز به تایید دارد)
- جستجو در آگهی‌های فعال و مشاهده جزئیات
- نشان کردن آگهی (علاقه‌مندی‌ها)
- گفتگوی خریدار و فروشنده درباره هر آگهی (بدون امکان پیام به آگهی خود)
- امتیازدهی ۱ تا ۵ به فروشنده (فقط خریداری که گفتگو داشته، فقط یک بار برای هر آگهی)
- پنل مدیریت: بررسی آگهی‌ها، مدیریت دسته‌بندی‌ها و کاربران

### موارد امتیازی پیاده‌سازی شده

- چند تصویر برای هر آگهی (تا ۵ تصویر) با گالری در صفحه جزئیات
- دسته‌بندی دو سطحی (دسته مادر و زیردسته)
- مرتب‌سازی نتایج (جدیدترین، قدیمی‌ترین، ارزان‌ترین، گران‌ترین، بهترین امتیاز فروشنده)
- جستجوی ترکیبی (کلمه کلیدی + دسته + شهر + بازه قیمت + وضعیت کالا همزمان)
- داشبورد آمار برای مدیر (تعداد کاربران، آگهی‌ها به تفکیک وضعیت، گفتگوها و ...)
- مسدود کردن و رفع مسدودی کاربران توسط مدیر (کاربر مسدود نمی‌تواند وارد شود یا پیام بفرستد)
- نمایش آخرین پیام و تعداد پیام‌های خوانده‌نشده + علامت دیده شدن پیام (✓ و ✓✓)
- کامنت‌گذاری و JavaDoc برای کلاس‌ها و متدهای اصلی

## ساختار پروژه

```
backend/
  src/main/java/com/secondhand/backend/
    model/         کلاس‌های موجودیت (User, Advertisement, ...)
    repository/    رابط‌های JPA برای کار با دیتابیس
    service/       منطق اصلی برنامه
    controller/    مسیرهای REST API
    security/      JWT و بررسی دسترسی
    dto/           کلاس‌های ورودی درخواست‌ها
    config/        داده‌های اولیه (Seeder)
    exception/     مدیریت یکپارچه خطاها
  src/test/java/   تست‌های واحد

frontend/
  src/main/java/com/secondhand/frontend/
    MainApplication.java   نقطه شروع برنامه
    ApiClient.java         ارسال درخواست‌های HTTP
    SessionManager.java    نگهداری توکن و کاربر فعلی
    view/                  صفحه‌های برنامه (ورود، ثبت‌نام، فروشگاه، ...)
```
